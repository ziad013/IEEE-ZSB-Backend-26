
// UNION vs UNION ALL

UNION :
Duplicate Handling --> removes duplicate rows

Performance --> Slower than UNION ALL because the database has to sort or compare rows to eliminate duplicates


UNION ALL :
Duplicate Handling --> Keeps all rows even duplicates

Performance --> Faster than UNION because it does not remove duplicates
