# Question 1

DBMS --> Database Management System
-used to store data and organise it.
-lower security.
-used for small projects.
-like Microsoft Access.


RDBMS --> Relational Database Management System
-tybe of DBMS.
-store data but in related table using primary key and foreign key.
-higher security.
-used for large and complex projects.
-like MySQL.


# Question 2

DDL --> Data Definition Language
-Deal with structure of the database.
-work on table.
-like Create and Alter.


DML --> Data Manipulation Language
-Deal with real data.
-work on rows in the table.
-like Insert and Update.
